## EMBED_WHITELIST

Comma-separated list of allowed hostnames for embeds. Supports `*.example.com` as a subdomain wildcard. Use `*` to allow all embeds (potentially insecure)

`EMBED_WHITELIST`
* is optional
* type: `string`
* defined in this schema

### EMBED_WHITELIST Type
